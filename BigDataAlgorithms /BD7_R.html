1 The Validation Set Approach

We will use Auto data set from ISLR package. ISLR package must be downloaded the first time using install.packages(“ISLR”)

library (ISLR)
set.seed(1)#Note 1

lm.fit<-lm(mpg~horsepower,data=Auto,subset=train)

4 The Bootstrap

The Portfolio data set in the ISLR package.

alpha.fn=function(data,index){#Note 1
  X=data$X[index]
  Y=data$Y[index]
  return((var(Y)-cov(X,Y))/(var(X)+var(Y)-2*cov(X,Y)))}
#Note1: we create a function, alpha.fn()

Estimate alpha using all 100 observations

alpha.fn(Portfolio, 1:100)
## [1] 0.5758321
alpha.fn(Portfolio,sample(100,100,replace=T))
## [1] 0.5606514

Bootstrap analysis using the boot() function.

boot(Portfolio,alpha.fn,1000)# we produce R = 1,000 bootstrap estimates for α

  return(coef(lm(mpg~horsepower,data=data,subset=index)))
